# Learning Playwright

This repository contains practice files for learning Playwright and JavaScript basics.

## Contents

- `chapter_01_Basics/` — Basic JavaScript examples and Playwright setup verification.
- `chapter_02_Identifier_Literals/` — Notes and shortcuts related to identifiers, literals, and VS Code.
- `chapter_03_operators/` — Practice examples for JavaScript operators and related tasks.
- `chapter_04_if_else/` — Conditional logic exercises using if/else statements and test-status scenarios.

## Usage

Open this repository in Visual Studio Code and run the examples or add new practice files.

## Notes

This repository is used for learning and experimentation with Playwright and editor workflows.
